FAQ

Common questions

Is this financial advice? +
No. NEXUS is an analytical tool — not financial advice, investment recommendations, or a solicitation to trade. All signals are generated by automated algorithms and statistical models. Trading gold carries significant risk of loss. Always verify signals independently before executing any trade. Past system performance is not indicative of future results.
What data sources does NEXUS use? +
Live gold price from Stooq and Twelve Data with multi-source failover. DXY computed in real-time from FloatRates FX data. US 10Y Yield direct from FRED (St. Louis Federal Reserve). WTI crude from Stooq futures. VIX and SPX from Yahoo Finance. COT data from the CFTC public API weekly. Economic calendar from ForexFactory. All fetched live — no manual updates required.
Do I need a Twelve Data API key? +
Not immediately. NEXUS uses a multi-source fallback chain that works without any API keys — you get live gold prices from public sources within seconds. For real OHLC candlestick data across all 6 timeframes (required for precise SMC analysis), you'll want to connect a Twelve Data key via the Settings modal. Twelve Data's free tier offers 800 API calls/day — sufficient for personal use. The Grow plan at $29/month covers full production use.
How does the ML engine work? +
NEXUS uses a genuine 10-tree Random Forest with bootstrap sampling and random feature subsets — not logistic regression or simple heuristics. The model trains on your completed trades, extracting an 8-feature vector (macro score, ADX strength, sweep detected, BOS/CHoCH, zone proximity, session quality, brain confidence, cascade score). Out-of-bag accuracy is computed automatically. The model retrains every 20 cycles as new trades close. Minimum 10 real trade outcomes required before the model activates.
Can I use NEXUS on mobile? +
Yes. NEXUS is fully mobile-responsive with a bottom navigation bar, touch-friendly chart with pinch-to-zoom, and a FAB navigation button. The app runs in any mobile browser. A PWA manifest is included — tap "Add to Home Screen" on iOS or Android for a native-app feel with offline capability.
What's the difference between NEXUS chart and TradingView? +
NEXUS has two chart modes. The NEXUS chart is our own canvas-based renderer with auto-drawn zones, FVGs, OBs, PDH/PDL, equal highs/lows, sweep markers, and EMA overlays — all computed from live OHLC data without any manual input. The TradingView widget (toggle button in the chart header) is available for users who want to manually mark additional structure on top of TradingView's ecosystem. You don't need a TradingView account to use the NEXUS chart.
Is my trading data private? +
Yes. Your trade journal, ML training data, account settings, and API keys are stored either in your browser's localStorage (local) or in a Supabase database with Row Level Security enforced — meaning no one else can access your data, including us. API keys (Twelve Data, OANDA) are stored locally in your browser only and are never sent to any server. The Cloudflare Worker proxies API calls so your keys remain private.
Can I cancel my subscription? +
Yes, at any time, with no penalties or questions. Click your avatar → Manage Subscription → Cancel. Your access continues until the end of the billing period you've already paid for. We don't pro-rate, but we don't charge you again either.
